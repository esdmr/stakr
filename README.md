## Instal

Ensure [Node.js][node] and [PNPM][pnpm] are installed at the
[specified minimum versions](#stakr). (Previous versions may work, they are not
tested.)

```sh
git clone https://github.com/esdmr/stakr.git
cd stakr
pnpm i
```
